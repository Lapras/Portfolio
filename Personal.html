Maroon Engine

As part of the Aggie Coding Club, I teamed up with other students in order to build a game engine from scratch. Including a tile-based map editor, physics engine, and support for mechanics of a sidescrolling platformer. I was part of the physics engine team and helped prototype the engine's collsion detection and resolution systems. Click below to access the project's github.

High School Projects

Throughout high school, I was involved in a few extracurriculars where I developed code outside of robotics. I developed several games in the unity engine in my schools Video Game Development club, as well as one for the Technology Student Organization, which was also where I participated in my first Cybersecurity Capture the Flag competition.

Unity development

I've been developing video games using the Unity engine and the C# scripting language as personal projects since middle school. I've made a 2D platformer, a Galaga-style arcade game, and more recently a physics-based maze game with randomly generated maps. Alongside the Unity Engine I use Pixel art programs to create artwork and animations for the games. Unfortunately, this stuff has been a little lost to time because they were not linked into a github.
